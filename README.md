# Spine Anomaly Classification (Binary)

## Описание проекта

Данный проект посвящён задаче бинарной классификации патологий позвоночника на основе биомеханических и анатомических параметров таза и поясничного отдела позвоночника.

Цель — сравнить различные подходы машинного обучения и показать, как работа с распределениями, мультиколлинеарностью и понижение размерности (PCA) влияет на качество классификации.

---

## Данные

Набор данных содержит 310 наблюдений и следующие признаки:

| Признак                    | Описание                               |
| -------------------------- | -------------------------------------- |
| pelvic_incidence           | Угол наклона таза                      |
| pelvic_tilt                | Наклон таза                            |
| lumbar_lordosis_angle      | Угол поясничного лордоза               |
| sacral_slope               | Наклон крестца                         |
| pelvic_radius              | Радиус таза                            |
| grade_of_spondylolisthesis | Степень спондилолистеза                |
| Class_att                  | Целевая переменная (Normal / Abnormal) |

Целевая переменная была приведена к бинарному формату.

---

## Первичный анализ и EDA

В ходе разведочного анализа данных было выявлено:

* существенное различие распределений признаков между классами
* наличие выбросов, особенно в признаке `grade_of_spondylolisthesis`
* выраженная асимметрия распределений
* сильная корреляция между угловыми параметрами таза

Для анализа использовались:

* описательные статистики
* гистограммы и boxplot с разбиением по целевому классу
* корреляционная матрица

---

## Мультиколлинеарность

Расчёт VIF показал критически высокую мультиколлинеарность:

* pelvic_incidence, pelvic_tilt и sacral_slope имеют VIF → ∞

Это делает применение линейных моделей в исходном пространстве признаков нестабильным.

---

## Используемые модели

### 1️ Random Forest

Использован как базовая нелинейная модель, устойчивая к:

* мультиколлинеарности
* выбросам
* нелинейным зависимостям

**Результаты:**

* Accuracy ≈ 0.79
* ROC-AUC ≈ 0.72
* Высокий recall для класса патологии, но ограниченная общая разделяющая способность

---

### 2️ Pipeline: RobustScaler + PCA + Logistic Regression

Последовательность обработки:

1. RobustScaler — устойчивое масштабирование при наличии выбросов
2. PCA — устранение мультиколлинеарности и выделение латентных факторов
3. Logistic Regression — интерпретируемая линейная модель

**Результаты:**

* Accuracy ≈ 0.87
* ROC-AUC ≈ 0.88
* Более сбалансированные precision / recall

---

## Интерпретация PCA

Анализ нагрузок главных компонент показал:

* **PC1** отражает общую выраженность патологии позвоночника
* **PC2** описывает анатомические размеры таза
* последующие компоненты связаны с компенсаторными и локальными изменениями

PCA позволил представить клинически взаимосвязанные признаки в виде независимых латентных факторов.

---

## Сравнение моделей

| Модель        | Accuracy | ROC-AUC | Особенности                                                     |
| ------------- | -------- | ------- | --------------------------------------------------------------- |
| Random Forest | ~0.79    | ~0.72   | Устойчив к выбросам, слабая интерпретируемость                  |
| PCA + LogReg  | ~0.87    | ~0.88   | Интерпретируемость, устойчивость, лучшая обобщающая способность |

---

## Выводы

* Простая линейная модель может превосходить ансамблевые методы при корректной подготовке данных
* Мультиколлинеарность существенно ухудшает работу моделей в исходном пространстве признаков
* PCA эффективно выделяет клинически осмысленные факторы
* Логистическая регрессия в пространстве главных компонент показала наилучшее качество классификации
